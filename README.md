# Superstore Sales Dashboard 📊

## Project Overview
This project analyzes retail sales data from the Superstore dataset using Python and data visualization techniques. The goal of this project is to discover business insights related to sales, profit, customers, products, and regional performance.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights
- Dashboard Preparation

---

# Dataset
Dataset Used:
- Sample Superstore Dataset

Dataset contains:
- Orders
- Sales
- Profit
- Customers
- Products
- Regions
- Shipping Details
